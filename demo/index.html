Effect 01 — Reveal

Reveal

Elements animate into view as they enter the viewport. Configurable direction, distance, duration, and easing.

←
From left
Slides in from the left with an elastic finish.
↑
From bottom
Default — enters from below with cubic easing.
→
From right
Slides in from the right with an elastic finish.
↓
From top
Falls down from above — good for nav items.
import { reveal } from 'scroll-craft' reveal('.card', { direction: 'left', distance: '40px', duration: 800, ease: 'elasticOut', })

Effect 04 — Text Reveal

Text Reveal

Split text into accessible word or letter spans and stagger each piece into view.

Words arrive with intent, one beat at a time.

Letters cascade too — while nested emphasis stays in the DOM instead of being flattened.

import { textReveal } from 'scroll-craft' textReveal('.headline', { type: 'words', stagger: 40 }) textReveal('.eyebrow', { type: 'letters', stagger: 18 })
Effect 05 — Stagger

Stagger

Observe a parent, reveal its children one after another with a configurable delay.

01

Resolve once

Children are selected before observing, so layout is never measured during frames.

02

Sequence cleanly

Each child starts after the configured delay, creating a cascading reveal.

03

Clean up

Returns a cleanup function, tracked through destroy() on the class API.

import { stagger } from 'scroll-craft' stagger('.feature-grid', { children: '.card', direction: 'up', stagger: 80, duration: 600, })

Blur Reveal

Effect 07 — Zoom

Zoom

Scale elements from a smaller starting point while fading them into view.

01

Composited scale

Opacity and transform stay on the compositor path for smooth card and media reveals.

02

Centered origin

Sets transform-origin to center while running, then restores styles on cleanup.

import { zoom } from 'scroll-craft' zoom('.product-card', { from: 0.94, duration: 600, ease: 'cubicOut', })

Effect 10 — Scene

Scene

Scrub a section timeline from progress 0 to 1 as the section top travels through the viewport.

Scroll-driven story beats.

The title fades in, the image slides upward, and this card rotates into alignment from the same section progress value.

SC
import { scene } from 'scroll-craft' scene('.story', [ { target: '.title', from: { opacity: 0 }, to: { opacity: 1 } }, { target: '.image', from: { y: 80 }, to: { y: 0 } }, { target: '.card', from: { rotate: -8 }, to: { rotate: 0 } }, ])
