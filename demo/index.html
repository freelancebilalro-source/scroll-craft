Effect 01

Reveal

Elements animate into view as they enter the viewport. Configurable direction, distance, duration, and easing.

↑
From left
Slides in from the left side of the viewport.
↕
From bottom
Default direction — enters from below with cubic easing.
→
From right
Slides in from the right with an elastic finish.
↓
From top
Falls down from above — good for navigation items.
import { reveal } from 'scroll-craft' // Simple reveal('.card') // Configured reveal('.card', { direction: 'left', distance: '40px', duration: 800, ease: 'elasticOut', delay: 120, })

API

Chainable class

All three effects are also available on a single ScrollCraft instance. Call .destroy() to clean up.

import { ScrollCraft } from 'scroll-craft' const sc = new ScrollCraft() sc.reveal('.hero-text', { delay: 100, direction: 'up' }) .counter('[data-count]', { duration: 1200 }) .progress('#timeline .step') // Later — disconnect all observers + listeners sc.destroy()
